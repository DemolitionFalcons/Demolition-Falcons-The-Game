Demolition Falcons API

A an imaginary game of type ladders and snakes.

Demolition Falcons API

  • GET a game - api/game
  • GET a game by Id - api/game/{id}
  • GET all characters - api/character
  • GET a character by name - api/character/{name}
  • GET available maps - api/map
  • GET map by name - api/map/{name}
